Sobre GitHub e Git

-Git: tecnologia de versionamento (histórico de código)
-GitHub: site/rede social de código

Uso geral:
-Linha de Comando (CMD, Terminal, PowerShell)
-GitHub Desktop: software do GitHub para manipulação de projetos Git


Para uso em casa, instalar:

1) Git: https://git-scm.com/downloads
2) GitHub Desktop: https://desktop.github.com/
